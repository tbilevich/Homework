DataContainer - класс реализации методов задания

IntComparator - класс реализующий интерфейс компаратор для типа элементов Integer

StringAlphabeticComparator - класс реализующий интерфейс компаратор для типа элементов String(в алфавитном порядке)

StringLengthComparator - класс реализующий интерфейс компаратор для типа элементов String(по длине строки)

Runner - стартовый класс для запуска программы
